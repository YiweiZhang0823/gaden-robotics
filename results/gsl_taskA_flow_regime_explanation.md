# 任务 A：三类场景为何分别视为 diffusion-dominant、time-varying 和 advection-dominant

## 任务 A 的目标

这一部分对应导师提到的第一点：从物理角度说明为什么某个场景更偏 `diffusion-dominant`，某个场景更偏 `advection-dominant`，并说明 `exp2` 为什么介于两者之间。

这里不引入新的理论概念，也不推导复杂公式，而是采用：

- 物理直觉解释
- 现有实验结果作为支撑
- 少量可观察指标作为辅助证据

也就是说，这部分是“带证据的定性判断”，而不是严格建模。

## 判定思路

我主要用四类现象来判断一个场景更偏扩散还是更偏平流：

1. 羽流空间范围是否宽
2. 上游或羽流外位置是否还能获得气味线索
3. 算法是否容易进入 `SURGE` / `SPIRAL`
4. 同一初始状态的结果是否表现出明显间歇性和随机性

这四类现象背后的物理意义是：

- 如果气体在空间中分布较宽，风向约束不强，那么更接近 diffusion-dominant。
- 如果气体主要沿主风方向形成狭窄羽流，上游或羽流外几乎没有气味，那么更接近 advection-dominant。
- 如果羽流会随时间摆动、同一状态时而成功时而失败，则更接近 time-varying / intermediate。

## 为什么 `exp1_diffusion` 视为 diffusion-dominant

### 物理直觉

`exp1` 的主要特征是弱/无平均风，气味不是被强主风拉成一条很窄的羽流，而是在气源附近更弥散地扩散开来。

因此，在这个场景中：

- 羽流边界不那么“锋利”
- 上游和侧向位置并不会完全失去气味
- 机械臂即使没有对准羽流中心，也较容易接触到可用气味

### 实验支撑

阶段一中，`exp1_diffusion` 的五类初始状态全部成功：

| 初始状态 | success | time_s | final_raw | final_confidence |
| --- | ---: | ---: | ---: | ---: |
| `far_off_plume` | 1 | 27.0 | 44.574 | 0.925 |
| `far_near_edge` | 1 | 26.8 | 138.182 | 0.918 |
| `far_inside_plume` | 1 | 29.8 | 339.636 | 0.958 |
| `near_source_outside_plume` | 1 | 25.4 | 82.448 | 0.926 |
| `near_source_upwind` | 1 | 9.4 | 278.618 | 0.628 |

最关键的是：

- `near_source_upwind` 也成功，说明“上游侧完全闻不到气味”这件事在 `exp1` 中并不成立；
- `far_off_plume` 和 `near_source_outside_plume` 也成功，说明羽流外状态并没有受到很强惩罚；
- 五类状态全部成功，说明初始状态差异被明显弱化。

### 结论

因此，`exp1` 可以被合理地解释为：

> 气味分布以扩散效应为主，风向约束较弱，因此更接近 diffusion-dominant。

## 为什么 `exp3_advection` 视为 advection-dominant

### 物理直觉

`exp3` 的主要特征是存在明显定常主风，风向大致为 `+x`。这会把气体输运成一条较窄、方向性较强的羽流。

因此，在这个场景中：

- 是否位于羽流边缘或羽流内部非常关键
- 上游位置即使靠近气源，也可能几乎没有气味
- 羽流外位置即使横向距离不远，也难以获得 plume contact

### 实验支撑

阶段一中，`exp3_advection` 五类状态只有一类成功：

| 初始状态 | success | min_distance_m | final_raw | CAST/SURGE/SPIRAL |
| --- | ---: | ---: | ---: | --- |
| `far_inside_plume` | 0 | 0.087 | 0.000 | 128 / 33 / 0 |
| `far_near_edge` | 1 | 0.236 | 9.620 | 65 / 38 / 36 |
| `far_off_plume` | 0 | 0.715 | 0.000 | 162 / 0 / 0 |
| `near_source_outside_plume` | 0 | 0.534 | 0.000 | 162 / 0 / 0 |
| `near_source_upwind` | 0 | 0.170 | 0.000 | 162 / 0 / 0 |

最关键的证据有两个：

1. `near_source_upwind` 离气源最近，但仍失败，且 `final_raw=0`、`SURGE=0`
2. `near_source_outside_plume` 靠近气源但在羽流外，同样 `final_raw=0`、`SURGE=0`

这说明在 `exp3` 中：

- “离气源近”并不足以保证有气味线索
- 是否在主风带出的羽流路径上才是决定性因素

唯一成功的 `far_near_edge` 也进一步说明了这一点：

- 它虽然距离气源更远
- 但因为位于羽流边缘，能获得弱但有效的 plume cue
- 因而进入 `SURGE` 和 `SPIRAL` 并最终成功

### 结论

因此，`exp3` 可以被合理地解释为：

> 气体输运主要受主风控制，羽流较窄且方向性强，因此更接近 advection-dominant。

## 为什么 `exp2_time_varying` 介于两者之间

### 物理直觉

`exp2` 不是像 `exp1` 那样几乎无风，也不是像 `exp3` 那样有稳定窄羽流，而是存在随时间变化的风场。

这意味着：

- 羽流仍然受到风向影响，但不会固定成一条稳定窄带
- 上游或羽流外状态有时能短暂获得气味
- 同一个初始状态，可能因为瞬时羽流摆动而成功，也可能失败

### 实验支撑

阶段一中，`exp2_time_varying` 的主表结果是 3/5 成功：

| 初始状态 | success | min_distance_m | final_confidence | 说明 |
| --- | ---: | ---: | ---: | --- |
| `far_off_plume` | 0 | 0.252 | 0.000 | 主表 timeout，但重复试验曾成功 |
| `far_near_edge` | 0 | 0.215 | 0.002 | 靠近过气源，但确认不稳定 |
| `far_inside_plume` | 1 | 0.207 | 0.616 | 成功 |
| `near_source_outside_plume` | 1 | 0.245 | 0.810 | 成功 |
| `near_source_upwind` | 1 | 0.200 | 0.621 | 成功 |

这里最重要的现象不是单纯“有几组成功”，而是：

- `near_source_upwind` 能成功，说明上游侧并非稳定无气味；
- `far_off_plume` 同一状态一次失败、一次成功，说明羽流存在明显间歇性；
- `far_near_edge` 能靠近气源但不稳定确认，说明边缘气味线索会出现但不持续。

### 结论

因此，`exp2` 最合适的解释不是纯 diffusion，也不是纯 advection，而是：

> 风场与扩散共同作用，羽流既有方向性又有摆动和间歇，因此更适合作为 time-varying / intermediate 场景。

## 三个场景的对比结论

把三者放在一起，可以得到下面这个更适合汇报的判断：

| 场景 | 更合适的物理解释 | 主要证据 |
| --- | --- | --- |
| `exp1_diffusion` | diffusion-dominant | 五类状态均成功，上游和羽流外位置仍有明显气味 |
| `exp2_time_varying` | time-varying / intermediate | 同一状态可成可败，边缘和羽流外状态表现随机 |
| `exp3_advection` | advection-dominant | 上游/羽流外几乎无气味，只有羽流边缘状态稳定成功 |

也可以把它再压缩成一句话：

> `exp1` 中扩散效应更强，所以羽流覆盖范围更宽；`exp3` 中平流效应更强，所以羽流更窄且方向性更强；`exp2` 处在两者之间，因为风场随时间变化，羽流既受风向影响又具有明显间歇性。

## 这一部分怎样服务后续工作

任务 A 的意义在于：后续谈参数敏感性时，不再只是说“某个参数好不好”，而是可以进一步解释：

- 为什么某些参数在 `exp3` 中更关键
- 为什么某些参数在 `exp1` 中反而没那么敏感
- 为什么 `exp2` 中最容易出现不稳定、随机性和 false positive

也就是说，这一部分为后续“wind-dependent 参数差异”提供了风场层面的物理背景。

## 可直接向导师汇报的版本

> 我没有额外引入新的理论公式，而是从现有实验现象出发，对三个场景的主导机理做了带证据的定性判断。`exp1` 中五类初始状态都能成功，上游和羽流外位置也能获得明显气味，因此更接近 diffusion-dominant；`exp3` 中只有羽流边缘状态成功，而上游和羽流外位置即使靠近气源也几乎没有气味输入，因此更接近 advection-dominant；`exp2` 则介于两者之间，因为同一初始状态可能成功也可能失败，说明羽流既受风向影响，又具有明显的时间变化和间歇性。这样后续在比较不同风场下的参数敏感性时，就能把参数差异理解为和风场类型、也就是 wind condition 相关的现象。
