# 任务一汇报稿：按风场类型与初始状态关系整理阶段一结果

## 汇报目的

本部分对应导师提出的第一项任务：先观察不同风场下羽流的大致形态，再设计多种机械臂初始状态，并分析这些初始状态如何影响气源定位算法表现。

这一部分的重点不是继续调参，而是回答：

1. 不同风场类型下，羽流形态有什么差异？
2. 五类初始状态分别代表什么物理关系？
3. 为什么相同类型的初始状态，在不同风场中会表现不同？

## 实验设计思路

### 三类风场

我选取了三个代表性场景：

| 场景 | 风场类型 | 羽流特点 | 本部分关注点 |
| --- | --- | --- | --- |
| `exp1_diffusion` | diffusion-dominant | 弱/无平均风，气味分布更宽、更弥散 | 初始状态影响是否较弱 |
| `exp2_time_varying` | time-varying / intermediate | 风向和羽流随时间摆动，气味间歇出现 | 同一初始状态是否具有随机性 |
| `exp3_advection` | advection-dominant | 定常主风明显，羽流更窄、更有方向性 | 相对羽流位置是否比相对气源距离更重要 |

### 五类初始状态

为了对应导师提出的“相对气源位置、相对羽流边界位置、相对风向位置”三类关系，我在每个场景中统一设置了五类初始状态：

| 初始状态 | 物理含义 |
| --- | --- |
| `far_off_plume` | 远离气源，且离羽流边缘也有一段距离 |
| `far_near_edge` | 远离气源，但接近羽流边缘 |
| `far_inside_plume` | 远离气源，但已经在羽流内部 |
| `near_source_outside_plume` | 靠近气源，但完全在羽流外 |
| `near_source_upwind` | 靠近气源，且位于气源上游一侧 |

因此，这五类状态不是简单的五个坐标，而是五种具有明确物理意义的初始条件组合。

## 统一实验条件

所有实验都使用同一套 `CAST/SURGE/SPIRAL` 流程，并保持以下条件一致：

| 项目 | 设置 |
| --- | --- |
| 气源位置 | `(1.0, 4.95, 0.7)` |
| Franka 基座位置 | `(0.8, 4.65, 0.0)` |
| `timeout_s` | `180.0` |
| `success_radius` | `0.25 m` |
| `required_confidence` | `0.6` |
| `goal_preset` | `tilt_x_90` |
| `search_heading_yaw` | `3.1416` |

这样可以保证不同实验之间主要只比较风场类型和初始状态关系，而不是混入额外变量。

## 三类风场下的结果整理

### 1. `exp1_diffusion`：扩散主导场景

`exp1_diffusion` 中五类初始状态均成功。

| 初始状态 | success | time_s | final_raw | final_confidence |
| --- | ---: | ---: | ---: | ---: |
| `far_off_plume` | 1 | 27.0 | 44.574 | 0.925 |
| `far_near_edge` | 1 | 26.8 | 138.182 | 0.918 |
| `far_inside_plume` | 1 | 29.8 | 339.636 | 0.958 |
| `near_source_outside_plume` | 1 | 25.4 | 82.448 | 0.926 |
| `near_source_upwind` | 1 | 9.4 | 278.618 | 0.628 |

这一结果说明：

- 扩散主导场景中，气味覆盖范围较宽；
- 即使末端位于上游参考侧或羽流外，仍可能接触到有效气味；
- 因此初始状态对成功率的影响较弱。

换句话说，在 `diffusion-dominant` 条件下，羽流的“空间惩罚”较小，初始位置差异被明显弱化。

### 2. `exp2_time_varying`：时变风场

`exp2_time_varying` 的表现介于 `exp1` 和 `exp3` 之间。

| 初始状态 | success | time_s | min_distance_m | final_confidence | 说明 |
| --- | ---: | ---: | ---: | ---: | --- |
| `far_off_plume` | 0 | 180.0 | 0.252 | 0.000 | 主表 timeout；重复试验曾成功 |
| `far_near_edge` | 0 | 180.2 | 0.215 | 0.002 | 靠近气源但 confidence 不稳定 |
| `far_inside_plume` | 1 | 63.2 | 0.207 | 0.616 | 成功 |
| `near_source_outside_plume` | 1 | 39.6 | 0.245 | 0.810 | 成功 |
| `near_source_upwind` | 1 | 16.6 | 0.200 | 0.621 | 成功 |

这一结果表明：

- 时变风场中，羽流会摆动和间歇出现；
- 因此相同初始状态不一定总得到相同结果；
- 初始状态的影响开始增强，但不是像平流主导场景那样固定、单调。

这里最重要的特征不是“谁成功谁失败”本身，而是：

> 时变风场会让边缘状态、羽流外状态和上游状态都具有一定随机性。

因此，`exp2` 更适合作为后续分析参数敏感性的中间场景。

### 3. `exp3_advection`：平流主导场景

`exp3_advection` 中，五类状态里只有 `far_near_edge` 成功。

| 初始状态 | success | time_s | min_distance_m | final_raw | CAST/SURGE/SPIRAL |
| --- | ---: | ---: | ---: | ---: | --- |
| `far_inside_plume` | 0 | 180.0 | 0.087 | 0.000 | 128 / 33 / 0 |
| `far_near_edge` | 1 | 156.0 | 0.236 | 9.620 | 65 / 38 / 36 |
| `far_off_plume` | 0 | 180.2 | 0.715 | 0.000 | 162 / 0 / 0 |
| `near_source_outside_plume` | 0 | 180.0 | 0.534 | 0.000 | 162 / 0 / 0 |
| `near_source_upwind` | 0 | 180.2 | 0.170 | 0.000 | 162 / 0 / 0 |

这一结果最能体现导师强调的“相对羽流和相对风向关系”：

- `near_source_upwind` 虽然离气源最近，但由于位于上游侧，几乎没有气味输入，最终失败；
- `near_source_outside_plume` 虽然靠近气源，但因为完全在羽流外，也无法进入有效搜索；
- `far_near_edge` 虽然离气源更远，但因为位于羽流边缘，能获得弱但有效的 plume cue，最终成功。

因此，在 `advection-dominant` 场景中：

> 是否接触羽流边缘或羽流内部，比是否单纯靠近气源更重要。

## 风场类型如何改变初始状态影响

如果把三类风场放在一起看，可以得到下面的整体规律：

| 风场类型 | 初始状态影响强弱 | 主要原因 |
| --- | --- | --- |
| diffusion-dominant | 弱 | 羽流宽，气味覆盖范围大，羽流外或上游位置仍可能接触气味 |
| time-varying | 中等且不稳定 | 羽流随时间摆动，结果受瞬时羽流形态影响 |
| advection-dominant | 强 | 羽流窄且方向性强，必须接触羽流边缘或内部才更容易成功 |

也就是说，阶段一结果支持如下判断：

1. 初始状态对算法表现的影响是真实存在的；
2. 这种影响不是固定不变，而是取决于风场类型；
3. 风场越接近平流主导，初始位置相对羽流边界和风向的作用越明显。

## 任务一的核心结论

本部分可以归纳为一句更适合汇报的结论：

> 阶段一结果表明，初始末端位置对气源定位表现的影响并不是固定不变的，而是随风场类型改变。在扩散主导场景中，羽流较宽，初始状态影响较弱；在时变风场中，羽流具有间歇性和随机性，相同初始状态可能产生不同结果；在平流主导场景中，羽流较窄且方向性强，末端是否接触羽流边缘或位于羽流内部，比单纯距离气源远近更关键。

## 这部分如何衔接到下一阶段

任务一的意义不只是说明“初始状态有影响”，更重要的是为后续超参数分析提供物理背景。

具体来说：

- `exp1_diffusion` 说明，在扩散主导场景中，算法对初始状态较不敏感；
- `exp2_time_varying` 说明，在时变风场中，边缘和羽流外状态具有随机性；
- `exp3_advection` 说明，在平流主导场景中，是否存在 plume contact 是决定性因素。

因此，后续再去分析 `required_confidence`、`threshold`、`plume loss` 等超参数时，就不是脱离物理背景的调参，而是：

> 在不同风场和不同初始状态关系下，分析哪些参数更敏感，以及为什么敏感。

## 可直接口头汇报的版本

> 我先完成了三种风场下五类初始状态的实验，并把这些状态重新解释成相对气源、相对羽流边界和相对风向三类关系。结果表明，在扩散主导场景中，羽流更宽，初始状态影响较弱；在时变风场中，羽流具有明显间歇性，同一状态可能成功也可能失败；在平流主导场景中，是否接触羽流边缘或羽流内部比单纯离气源近更重要。这说明初始状态对算法表现的影响是真实存在的，而且这种影响取决于风场类型。这样就为后续分析不同风场下的参数敏感性提供了物理背景。
